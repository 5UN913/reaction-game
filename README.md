# 反应游戏 - Three.js增强版

## 项目概述

反应游戏是一款基于Three.js的3D视觉效果增强的反应速度测试游戏，展示前端技术能力和用户体验设计。

- **主要目的**：测试用户的反应速度和准确性，同时展示Three.js的3D视觉效果和前端技术能力
- **目标用户**：潜在雇主、同行开发者、潜在客户、教育机构和个人品牌建设受众
- **技术亮点**：HTML5 Canvas、JavaScript动画、Three.js 3D视觉效果、响应式设计

## 技术栈

- **前端**：原生JavaScript + Three.js + CSS3
- **3D库**：Three.js (通过CDN引入)
- **字体**：Inter (通过Google Fonts引入)
- **存储**：localStorage (用于存储历史最高分)

## 核心功能

### 1. 游戏界面
- **开始界面**：游戏标题、开始按钮、规则说明
- **游戏界面**：3D目标显示、计时器、分数、剩余回合
- **结束界面**：最终得分、历史最高分、重新开始

### 2. 3D效果
- **3D目标对象**：随机生成的立方体、球体、圆柱体
- **粒子效果**：成功/失败反馈时的粒子爆炸
- **3D动画**：目标对象的旋转和浮动动画
- **深度和层次感**：通过3D空间增强视觉体验

### 3. 游戏机制
- **目标生成系统**：随机生成3D目标对象
- **反应检测**：点击/触摸事件检测
- **计时系统**：记录反应时间和游戏总时间
- **计分系统**：基于反应速度和准确率计算分数

## 如何运行

1. 克隆或下载项目到本地
2. 直接在浏览器中打开 `index.html` 文件
3. 无需安装任何依赖，所有资源通过CDN引入

## 游戏规则

1. 点击屏幕上出现的3D目标
2. 反应速度越快，得分越高
3. 游戏共10个回合
4. 点击空白区域会扣分

## 响应式设计

- 适配不同屏幕尺寸，包括桌面端、平板和移动设备
- 支持触摸事件，确保在移动设备上的良好体验
- 横屏和竖屏模式都能正常显示

## 性能优化

- 使用requestAnimationFrame进行动画循环
- 合理控制3D对象数量和复杂度
- 避免不必要的计算和渲染
- 使用对象池管理粒子效果，减少内存开销

## 技术实现亮点

1. **Three.js 3D场景搭建**：创建了完整的3D场景，包括相机、灯光、渲染器和3D对象
2. **射线投射碰撞检测**：使用Raycaster实现2D点击到3D对象的映射
3. **粒子系统**：实现了成功和失败时的粒子爆炸效果
4. **游戏状态管理**：清晰的游戏状态控制，包括开始、游戏中、结束三个状态
5. **响应式设计**：适配不同屏幕尺寸，提供良好的跨设备体验
6. **性能优化**：确保3D渲染流畅，至少30fps
